# Nexa Mortgage Website  (Volunteer Project)

A modern, responsive mortgage website built as part of my **volunteer work for a company**.  
The project showcases professional design, interactive features, and smooth animations to create an engaging and user-friendly mortgage experience.  

---

## 🌟 Project Overview  

During my volunteer work, I contributed to the development of this website to support the company’s mission of providing clear, accessible mortgage information online. My focus was on creating a **responsive, interactive, and professional platform** that works seamlessly across devices.  

---

## 🚀 Features  

### Design & UI  
- **Modern Design:** Clean, professional layout with gradient backgrounds and modern typography  
- **Responsive Layout:** Fully responsive design that adapts to desktop, tablet, and mobile  
- **Smooth Animations:** CSS transitions for a polished experience  
- **Interactive Elements:** Hover effects, smooth scrolling, and dynamic content  

### Navigation  
- **Fixed Navigation Bar:** Sticky header with backdrop blur effect  
- **Mobile Menu:** Hamburger navigation for smaller screens  
- **Smooth Scrolling:** Animated section transitions  
- **Active States:** Visual cues for current section  

### Sections  
- **Hero Section:** Landing area with call-to-action buttons  
- **Features:** Highlighting key benefits with animated icons  
- **Loan Programs:** Information on different mortgage types  
- **Mortgage Calculator:** Real-time, interactive payment calculator  
- **About Us:** Company info with animated counters and statistics  
- **Contact Form:** Inquiry and contact section with validation  
- **Application Form:** Mortgage application submission form  
- **Footer:** Comprehensive site information and quick links  

### Interactive Features  
- **Mortgage Calculator:** Real-time payment calculation updates  
- **Form Validation:** Input checks and user feedback  
- **Notifications:** Success and error messaging  
- **Loading States:** Button loading animations  
- **Counter Animations:** Animated statistics counters  
- **Parallax Effects:** Subtle scrolling effects

## 🚀 Getting Started  

### Prerequisites  
- A modern browser (Chrome, Firefox, Safari, Edge)  
- Local server (optional for development)  

### Installation  
```bash
# Clone the repository
git clone https://github.com/your-username/nexa-mortgage-website.git

# Open the project folder
cd nexa-mortgage-website

# Open index.html in your browser
```

### File Structure  
```
nexaMortgage/
├── index.html       # Main HTML file
├── styles.css       # CSS styles and animations
├── script.js        # JavaScript functionality
└── README.md        # Documentation
```

---

## 🔧 Customization  

### Colors  
Update CSS variables in `styles.css`:  
```css
:root {
  --primary-color: #2563eb;
  --secondary-color: #7c3aed;
  --success-color: #10b981;
  --error-color: #ef4444;
}
```

### Content  
- Add new `<section>` elements in `index.html`  
- Expand loan programs with new cards  
- Update calculator logic in `script.js`
